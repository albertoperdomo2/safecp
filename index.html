a simple background service to sanitize sensitive clipboard contents (like emails, tokens, IDs) on macOS and Linux

installation

clone the repo and run the install script:

git clone https://github.com/albertoperdomo2/safecp.git
cd safecp
./install

usage

after installation, safecp runs in the background and monitors your clipboard for sensitive content:

# copied token:
gho_1234abcd...

# automatically replaced with:
DUMMY_TOKEN_1

default patterns live at ~/.safecp.patterns.json and are fully customizable

features

  • runs automatically as a background service
  • detects and replaces secrets in real time
  • supports custom regex patterns
  • macOS (LaunchAgent) and Linux (systemd) support
  • minimalist and easily auditable
